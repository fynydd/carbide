Short Codes

The following are short codes (bits of text) you can enter into most any text input box to display data from your settings, like company name.

Short Codes

You can insert field data into your content by using short codes. These snippets of text use the format {{fieldName}} where "fieldName" is the name of the field in camel case format. For example, to insert the company street address, type:

{{companyStreetAddress}}

You can find field names by simply hovering your mouse over a caption in the left column. A tooltip will appear showing you the field name.

Note: short codes are available for fields in the home page node as well as the current content item being edited (and its parent, if it's nested content).

Utility Short Codes

Following the format above you can insert utility short codes for things like tags.

{{tags:Pages}} — Backoffice code for rendering a list of tags used for a given group.

CSS Custom Properties

The following are guides to customizing the appearance of the website as a whole, or individual components.

Global CSS Custom Properties

Below are the CSS custom properties with sample data for styling the overall site from within the settings area:

--background-color: var(--sf-color-white);
--foreground-color: var(--sf-color-white-contrast);

Navigation CSS Custom Properties

Below are the CSS custom properties with sample data for styling the navigation units:

--background-color: var(--sf-color-white);
--foreground-color: var(--sf-color-white-contrast);

--horizontal-unit-padding: 1.25rem;
--vertical-unit-padding: 3rem;

--background-color: #1b264f;
--foreground-color: #ffffff;
--first-max-width: 50%;
--sidebar-first-max-width: 75%;
--current-opacity: 0.5;
--item-gutter: 3rem;
--justify-content: flex-end;
--first-justify-content: flex-start;

Container Unit CSS Custom Properties

Below are the CSS custom properties with sample data for styling container units:

--background-color: var(--sf-color-white);

--horizontal-unit-padding: 1.25rem;
--vertical-unit-padding: 3rem;
